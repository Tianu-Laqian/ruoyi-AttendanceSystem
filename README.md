# 基于ruoyi开发的考勤管理系统

基于ruoyi开发的考勤管理系统，初步实现了普通员工考勤打卡、打卡情况自查，管理员查看修改普通员工考勤情况，考勤规则设置，考勤异常自动识别（每日23点自动检查）等。

## 前端

用Ruoyi-Vue3

## 后端

用Ruoyi-Vue-Fast

## sql

导入ry-vue.sql

## 形式

ruoyi二次开发，前后端分离，建议前端代码使用vscode运行，后端代码使用idea运行。
